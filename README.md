# Mahjabin Portfolio Website

A modern responsive personal portfolio website built using HTML, Tailwind CSS, and JavaScript.

This portfolio showcases my projects, skills, experience, and creative frontend development work through a clean and interactive user experience.

---

# Live Features

- Modern Hero Section
- Animated UI Components
- Responsive Design
- Dynamic Project Details Page
- About Me Page
- Contact Page
- Resume / CV Download
- Smooth Scroll Animations
- Floating Background Elements
- Mobile Navigation Menu
- Interactive Project Showcase
- GitHub & Live Demo Buttons
- Professional Glassmorphism Design

---

# Built With

- HTML5
- Tailwind CSS
- JavaScript
- Font Awesome
- Google Fonts
- html2pdf.js
